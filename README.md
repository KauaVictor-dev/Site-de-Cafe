☕ Coffee Shop Website
📌 Sobre o Projeto

Este é um site de cafeteria desenvolvido com foco em design moderno, usabilidade e experiência do usuário. O projeto simula um ambiente real de e-commerce de café, apresentando produtos, navegação intuitiva e funcionalidades básicas.

🚀 Tecnologias Utilizadas
HTML5
CSS3
JavaScript
🎯 Funcionalidades
🛍️ Listagem de produtos (cafés)
🛒 Carrinho de compras
🔎 Navegação por seções
📱 Design responsivo
✨ Interface moderna e interativa
💡 Objetivo

Praticar desenvolvimento front-end, manipulação de DOM e criação de interfaces atrativas.

🧠 Aprendizados
Manipulação de elementos com JavaScript
Organização de layout com CSS
Estruturação de páginas com HTML
